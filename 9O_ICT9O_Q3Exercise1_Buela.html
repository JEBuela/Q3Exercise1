HTML

HyperText Markup Language

An HTML or Hyper Text Markup Language is the structure of a web page. This is why HTML is very important in the internet because without this, our internet content and all components in a website wouldn't exist. This is defined as the main standard markup language for making web pages. When we use HTML, we have the ability to create our own websites.

CSS

Cascading Style sheets

The CSS or the Cascading Style Sheets is used to add design and add style to a website. This contains the different variations such as the display for different devices, screen sizes, designs and layouts. The CSS is one of the components of the HTML, it helps the HTML web page look presentable and appealing to the eyes.

Bootstrap

Bootstrap Version 4

We are able to easily construct responsive designs due to Bootstrap. We can agree that using Bootstrap is incredibly simple. This is a free font-end framework designed to make web development quicker and simpler. Bootstrap's emphasis on mobile devices and cross-browser interoperability are advantages.
